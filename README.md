# StarSlime

「星のネバール」

（概要・テーマ）
この作品は、2019年の日本ゲーム大賞に応募する為に開発した作品です。
今回の日本ゲーム大賞のテーマは「☆」だったのと、
日本ゲーム大賞で大賞を貰える作品にパズルゲームが多い為、
星のかけらを集めるパズルゲームというテーマで制作しました。

(開発時期・期間)
HAL東京4年生時の2019年4月～2019年6月(3ヶ月)

（開発ツール）
・ゲームエンジン：Unreal Engine 4(4.21.2)
・3Dデータ制作：Maya(2017 アップデート5)
・2Dデータ制作：Substance Painter(2018.1.1)・デザイナー(2018.1.2)
・マネジメントデータ制作：PowerPoint2016・Excel2016・Word2016
・ミュージックデータ制作：Studio One 4 Prime(4.1.3)

（チームメンバー）
開発チーム（計9人）
・プログラマー３人（担当）
・プランナー１人
・デザイナー４人
・サウンド １人

（担当内容）
自分はディレクターとプログラマーの 2 つを兼任しました。
プログラムはタイトル画面とジャンプマシーン・天秤などのギミック物をブループリントで制作しました。
マネジメントはチームメンバーのスカウトをしました。

（ゲームシステム）
ステージに散らばっている星のかけらを全て集めてステージクリアーを目指すパズルアクションゲームです。
・星のかけらを石碑に全てはめると次のステージへ行けて全てのステージをクリアーするとゲームクリアーになります。
・スライムはE・Qキーで伸ばす事ができ、キューブを移動させる事ができます。
・ジャンプマシーンに乗ると高く飛ぶ事ができます。
・天秤に乗ると天秤が下がりもう片方の天秤が上がります。

(操作方法)
[キーボード]
・W・A・S・Dでキャラクターの移動をします。
・EキーまたはQキーを押すことでスライムを伸ばしキューブを引き寄せる事ができます。
・スペースキーでジャンプする事ができます。
・Oキーで星のかけらを取ります。
・Pキーでポーズ画面を開くことができ、Jキーで決定します。

[xbox360コントローラー]
・左スティック：移動
・L2：左に伸びる
・R2：右に伸びる
・X：星のかけらを取る/離す
・A：ジャンプ
・B：ポーズ画面での決定
・スタート：ポーズ画面

(起動方法)
「星のネバール(ビルド)」というフォルダーを開いてください。
その中にある、「星のネバール」というアプリケーションをダブルクリックしてください。
